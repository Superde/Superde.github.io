A Representation Learning Framework for Property Graphs

摘要

图嵌入具有重大的研究价值，目前的图嵌入方法要么只注重图的拓扑结构，要么只注重节点属性，边属性。因此，本文提出PGE，将节点属性，边属性，图的拓扑结构结合的图嵌入方法。

介绍

PGE：PGE通过将拓扑和属性信息都结合到图嵌入过程中，采用了一种偏向方法来区分邻居和相应连接边的影响。

主要步骤：

（1） 基于属性的节点聚类，基于与节点的属性相似性将节点的邻域分为相似和不相似的组

（2） 偏置邻域采样以获得根据偏置参数（基于聚类结果设置）采样的较小邻域，从而使嵌入过程更具可扩展性

（3） 邻域聚合通过将采样邻域的属性信息与经过神经网络训练的权重矩阵进行聚合来计算最终的低维节点嵌入。

相关定义：

图定义：$\cal G=\{\cal V,\cal E,\cal P,\cal L\}$，$\cal V$代表节点集，$\cal E$代表边集

$\cal P$ 是所有属性的集合，$\cal P=\cal P_{\cal V}\bigcup \cal P_{\cal E}$，$\cal L$是所有标签的集合，$\cal L=\cal L_{\cal V}\bigcup \cal L_{\cal E}$。

$\cal N_v$为顶点 $v$ 的邻域顶点

相似度定义：两个顶点的相似度可分成两部分，属性相似度+拓扑结构相似度

PGE详细步骤

  1. 基于顶点属性聚类

    使用K-means或者DB-SCAN等聚类算法，根据顶点的属性进行聚类，生成 k 个聚类中心，

    C = { $C_1$ ,$C_1$,…,$C_k$ }

  2. 偏向邻域采样

    基于第一步的聚类结果进行一个有偏向的邻域采样：

    • 如果邻域节点$v’$和$v$属于同一个类，就分配一个偏向$b_s$给$v’$，表示这两个节点是相似的；否则就分配一个偏向$b_d$给$v’$，表示这两个节点不相似
    • 我们归一化对$N_v$的偏差，然后根据归一化的偏差对$N_v$进行采样，以获得固定大小（25）的采样邻域$N_v^s$

  3. 邻域聚合（类似GraphSAGE中的Mean Aggregator）

    基于第二步获得的$N_v^s$，我们通过乘以神经网络训练的权重矩阵来汇总其属性信息以获得$z_v$。

    $P_v$是节点v的元素属性向量，A()表示的向量连接操作，$W^K$是参数矩阵

  4. 考虑边属性的邻域聚合

    对采样的图 G 考虑边的方向和边的属性，使用和节点属性处理相同的策略。如果边是有向边，则分别考虑in-edges和out-edges。
    将边分成$k^e$个cluster，$C^e=C_1^e,C_2^e,…C_{k^e}^e$，然后训练2✖️$k^e$个矩阵，{$W_1^1$,$W_2^1$,…$W_{k^e}^1$}和，最后根据下{$W_1^2$,$W_2^2$,…$W_{k^e}^2$}式计算$Z_u$。
